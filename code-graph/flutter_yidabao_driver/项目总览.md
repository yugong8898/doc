# flutter_yidabao_driver 项目总览

## 项目简介

易达宝司机端 Flutter 应用，同时支持两个品牌版本：
- **易达宝**（yidabao）：包名 `com.wanlianyida.driver`
- **万联通**（wanliantong）：包名 `com.wanlianyida.hig.app`

通过 Flutter Flavor 机制实现双品牌差异化构建，共用同一套代码逻辑。

- **Flutter 版本**：3.32.8（stable channel）
- **Dart SDK**：^3.8.1
- **当前版本**：1.0.22+51（生产）
- **状态管理**：GetX（get: ^4.7.2）

## 架构设计

### 整体架构

```
┌─────────────────────────────────────────────┐
│                  App Entry                   │
│  main_yidabao.dart / main_wanliantong.dart   │
└──────────────────────┬──────────────────────┘
                       │
┌──────────────────────▼──────────────────────┐
│              AppPackageConfig                │
│         品牌 Flavor 识别与路由分发             │
└──────────────────────┬──────────────────────┘
                       │
┌──────────────────────▼──────────────────────┐
│                   Pages                      │
│  GetX Controller + View + State 三层架构      │
└────┬──────────┬──────────┬──────────────────┘
     │          │          │
  Store      API层       Utils
  (状态)    (网络)       (工具)
```

### 状态管理（GetX）

项目使用 GetX 作为状态管理框架，遵循 Controller-View 分离模式：
- `logic.dart` / `controller.dart`：业务逻辑（等同于 GetxController）
- `view.dart`：UI 层（StatelessWidget + Obx）
- `state.dart`：状态数据（部分模块使用）

## 核心业务流程

### 启动流程

```
main_yidabao.dart
    └── AppPackageConfig.init(flavor: 'yidabao')  // 初始化品牌信息
    └── ApiService.init()                          // 初始化网络请求
    └── UserStore.init()                           // 恢复本地登录态
    └── SdkInitHelper.init()                       // 初始化三方SDK（个推、Firebase等）
    └── start_screen（启动广告页）
    └── TabBar（主 Tab 页）
```

### 登录流程

```
login_driver/
    ├── 短信验证码登录（SmsLogin）
    │     └── 发送验证码 → 输入验证码 → 获取 token → UserStore.saveToken()
    └── 一键登录（OneKeyLogin）
          └── OneKeyLoginHelper → 运营商授权 → 获取 token → UserStore.saveToken()
```

### 认证流程

```
certification/
    ├── identity_verification/    // 身份证实名认证（OCR + 活体）
    ├── drivers_license_certification/  // 驾驶证认证
    ├── driving_license_certification/  // 行驶证认证
    ├── qualification_certificate/      // 资质证书认证
    ├── road_transport_certificate/     // 道路运输证认证
    └── driver_car_certification_label/ // 车辆标签认证
```

### 运单业务流程

```
home_page/waybill_page/
    ├── waybill_list_page/         // 运单列表（全部/待处理/运输中/已完成/已取消）
    ├── waybill_detail_page/       // 运单详情
    ├── waybill_card_punching/     // 装卸货打卡（上传图片、位置）
    ├── waybill_pictures_edit_page/  // 运单图片编辑
    ├── waybill_appraise/          // 运单评价
    └── waybill_change_record/     // 运单变更记录
```

### 货源大厅流程

```
home_page/freight_hall_home/
    ├── freight_hall/             // 货源列表
    ├── freight_attention/        // 关注路线
    └── recommend_freight/        // 推荐货源
```

## 双品牌差异化

通过 `AppPackageConfig.isYiDaBao` / `AppPackageConfig.isWanLianTong` 在代码中区分两套品牌逻辑：

- **UI 差异**：不同 Logo、颜色主题、文案
- **H5 链接差异**：易达宝用 `10000da.vip` 系列域名，万联通用 `wanlianyida.com` 系列
- **WebView 消息处理**：`web_view_component/handler_message/yidabao/` vs `wanliantong/`
- **MethodChannel 通信**：通道名基于包名动态生成
- **资产目录**：`assets/wan_lian_tong/` 存放万联通专属资源
