# platformweb 项目总览

## 项目定位

**万联易达 — 无车承运人平台（PC 端 4PL）**

面向平台运营人员的后台管理系统，承担货运交易全流程的管理与监控职责，包括运单管理、司机/车辆/承运商信息审核、费用结算、税务对接、风控策略配置、信用额度管理等核心业务。

## 路由与权限体系

### 静态路由（`src/router/routers.js`）
- `/systemlogin` — 登录页
- `/404`、`/401`、`/redirect` — 功能页

### 动态路由（`src/router/index.js`）
路由在用户登录后动态生成，流程如下：

```
router.beforeEach
  ├── 未登录 → 重定向 /systemlogin
  └── 已登录 & 未加载菜单
        ├── store.dispatch('app/getUser')      ← 获取用户信息
        ├── store.dispatch('app/getUserMenu')  ← 获取菜单树
        └── filterAsyncRouter(datas)
              ├── 遍历后端返回的功能列表
              ├── 按 actionName 去重（funcType=1 页面优先）
              ├── loadView(component) 懒加载组件
              ├── 将按钮权限（funcType=2）注入 route.meta
              └── router.addRoutes(asyncRouter)
```

### 按钮权限
按钮权限以 `styleName` 为 key 存储在 `route.meta` 中，页面内通过以下方式获取：

```javascript
Object.keys(this.$route.meta).forEach(v => {
  this.$set(this.controlBtn, v, this.$route.meta[v])
})
```

## HTTP 请求层（`src/utils/request.js`）

基于 Axios 封装，核心行为：

**请求拦截器：**
- 调用弱网检测 `throttledCheckNetwork()`
- Token 一致性检查（多标签页 token 变更时自动登出）
- 注入 `Authorization` header
- 通过 `@wlydfe/http` 的 `addHttpCommonData` 添加公共字段（`passport`、`client_biz_type`）

**响应拦截器：**
- blob/arraybuffer 直接返回（文件下载）
- GET 请求直接返回 `response.data`（导出场景）
- `succeed: false` 时处理错误码：
  - `ACOM0004` + `isRefreshResponse` → 走 refreshToken 续签流程
  - 错误白名单（`100006`、`ACOM0009`）直接返回 errCode
  - 登出类错误码 → 触发 `doLogOut` + 页面刷新

## 认证与 Token 管理（`src/utils/auth.js`）

- Token Key：`EL-ADMIN-TOEKN-4PL`（配置于 `src/config/index.js`）
- 存储策略：优先写入 Cookie（`js-cookie`），同时同步写入 `localStorage` 作为 iframe 环境备用
- 读取策略：Cookie 优先，不存在则降级到 `localStorage`
- `refreshToken`：token 过期时（`ACOM0004` 错误码）通过 `src/utils/refreshToken.js` 续签，续签失败则登出

## 环境变量体系

环境变量统一在根目录 `.env.config.js` 中维护（不提交 git），由 `scripts/generate-env.js` 生成 Vue CLI 所需的 `.env.*` 文件：

| 脚本命令 | 作用 |
|----------|------|
| `npm run gen:env -- dev` | 生成开发环境变量 |
| `npm run check:env` | 生成全部环境并校验必填项 |
| `npm run dev` | 自动生成 dev 环境变量后启动 devServer |

本地代理目标（`DEV_PROXY_API` 等）维护在 `.env.config.js` 的 `devServerProxy` 节中，支持通过 `.env.development.local` 个人覆盖。
