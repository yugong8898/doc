## 一、查看结算信息（waybillSettlement/read）

### 查询参数
- `waybillPoolId` — 运单池ID（主键）
- `type` — 操作类型（`read`=查看）
- `waybillId` — 运单号

### 获取详情接口

**完整路径：**
```
POST /gateway/bms/BmsWaybillPool/viewDetailPool
```

请求体：
```json
{ "waybillPoolId": "xxx" }
```

**接口文件：** `src/api/userBase.waybillSettlement.js` → `postWaybillDetails()`

## 数据关联关系

```
运单 (waybillId)
  ├─ 运单池 (waybillPoolId)  →  BmsWaybillPool（查看结算信息）
  ├─ 对账单 (waybillAccountsId) →  BmsWaybillAccounts（司机对账详情）
  └─ 付款单 (id/paymentId)   →  BmsPayment（查看付款申请）
```

## 页面4：运单详情（match-owner-pc）

**接口：** `POST /lmt-user/waybill/detail`

**参数：** `{ waybillId }`

**核心字段：**

### 运输信息（transportInfoDTO）
- `driverName` - 司机姓名（田铃铃）
- `plateNo` - 车牌号（辽AAA123 重型厢式货车）
- `driverMobile` - 司机手机号（15732194273）
- `vehicleType` - 车型
- `vehicleLength` - 车长
- `transportMileage` - 预估里程（1.99km）
- `estArriveDate` - 预计送达时间
- `alias` - 货源别名
- `waybillAlias` - 运单别名

### 结算信息（accountInfoDTO）
- `invoiceType` - 业务类型（20-开票）
- `networkMainBodyName` - 网货公司
- `unitPrice` - 单价
- `pricingMethod` - 计价方式
- `settWeightType` - 结算重量指定
- `zeroRoundingMethod` - 自动抹零
- `inctaxFreightAmount` - 预计运费(含差价)
- `freightAmount` - 预计运费(不含差价)
- `difFreightAmount` - 运费差额总额
- `freightDiscountAmount` - 优惠券抵扣
- `taxRate` - 差价率

### 亏吨扣款（deductibleRule）
- `deductibleType` - 扣款类型（0-不扣款, 1-按比例, 2-按固定值）
- `deductibleValue` - 免赔系数
- `goodsLostValue` - 货损单价(元/吨)

### 其他信息（otherInfoDTO）
- `customerName` - 客户名称（大大用租赁有限公司）
- `remark` - 备注
- `imageList` - 图片列表

### 货物保障服务（insuranceInfo）
- `buyerName` - 购买方
- `guaranteeType` - 保障类型（10-定额, 20-货值）
- `unitValue` - 每吨货值/整车货值
- `guaranteeRate` - 保障服务费率
- `guaranteeServiceCharge` - 货物保障服务金额
- `quota` - 保障额度
- `insuredName` - 被保人

**其他接口：**

| `POST /lmt-user/waybill/freight-record` | 运费修改记录 |
