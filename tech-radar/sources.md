## 一、来源原则

每日技术雷达优先选择稳定、可追溯、可订阅的来源。默认以 RSS、官方博客、GitHub、主流技术社区为主。

优先级从高到低：

1. 官方博客和官方发布。
2. 高质量工程团队博客。
3. GitHub Trending / Release / Discussions。
4. 技术社区精选文章。
5. 聚合站摘要。
